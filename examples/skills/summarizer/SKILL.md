# Summarizer Skill

Use this skill when the user asks to summarize, condense, shorten, or extract the main points from text.

## Instructions

When applying this skill:

1. Read the user's text carefully.
2. Preserve the core meaning.
3. Remove repetition and filler.
4. Keep the summary shorter than the original.
5. Do not invent facts that are not present in the text.

## Output Format

Return the answer in this format:

```text
Summary:
<short summary>

Key Points:
- <point 1>
- <point 2>
- <point 3>
